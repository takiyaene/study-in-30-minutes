# 各回

## 第01回 (2021/01/25) .git/info/exclude について

### 何を実現するためのものか？
- Git で「自分だけが」追跡されたくないファイルを指定することができる
  - `.gitignore` の自分専用版

### どうやるか？
- 各リポジトリ配下の `.git/info/exclude` に追跡されたくないファイル名を書く
  - 正規表現を用いることができる

### 何が嬉しいか？
- 自分だけのスクリプトやコードを誤ってコミットしなくて済む
  - テストのために書いた一時的なコード
    - `my_test.rb`
    - `filterMethodTest.js`
  - エディタや導入しているツール特有の設定ファイル
    - `.idea/`
    - `.vscode/`
    - `.envrc`

### 発展
